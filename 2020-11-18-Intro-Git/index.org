* Ny Branch
Kopierer filer fra branch =origin/dev= for testing
- =git checkout --track origin/dev=
- =git pull=

* Eksisterende branch
Bytt branch =dev=
- =git checkout dev=
- =git pull=

* Slett endringene
Sjekk hvilken branch 
- =git branch=

Se hvis filen har endret
- =git status=

Tilbakestilt koder
- =git stash=
